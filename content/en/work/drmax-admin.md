E-shop administration platform for Dr.Max, a pharmacy group operating
across Central Europe: 70+ modules used daily by 150+ internal staff in
five countries. Platform-team work — technical standards, the shared
component library, architecture decisions, onboarding for 15–20 developers.

Currently leading the Vue 2 → Vue 3 + Nuxt migration, including the
automated migration script that carries it. Around the platform: order
tracking integrations for five shipping carriers in NestJS, a Go
microservice for media management, and the VitePress documentation that cut
consultation load from 8–12 hours a week to one.
